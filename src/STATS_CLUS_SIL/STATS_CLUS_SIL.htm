Silhouette Statistics for Cluster Analysis

This procedure produces silhouette statistics for cluster analysis results that may be useful in assessing the clustering results.

The procedure requires the cases to be in memory, and the execution time is proportional to the square of the number of cases. If you do not have the memory and patience for this, consider using a random sample of your data for this procedure.

Cluster Number Select the variable holding the cluster membership number. Typically the values range from 1 to K, where K is the number of clusters, however, gaps are allowed and will be treated as empty clusters.

Result Variables Enter names for variables to hold the number of the next best cluster for each case and for the silhouette value. Existing values, if any, will be overwritten.

• The silhouette value range is from -1 to 1 with values near 1 indicating that the case is well placed and small or negative values indicating that it is not.

Dissimilarity Measure Select the measure to be used in calculating the dissimilarities between cases and, hence, the silhouette scores. Normally this would be consistent with the measure used to calculate the clusters, but this procedure does not provide all of the many different clustering measures available in Statistics.

• All the clustering variables must be numeric except for the Gower measure.

Euclidean is standard Euclidean distance

Absolute Difference is the sum of the absolute values of the variable differences

Minkowski is the pth root of the sum of the pth power of the absolute differences. Enter the power to use below.

Gower distinguishes variables according to the measurement level. Scale variables are measured by absolute difference with each difference divided by the variable range. Ordinal variables are renumbered so that the values range from 1 to M, where M is the number of distinct values and then treated as scale. Nominal variable differences are 0 or 1 according to whether the values are equal or not.

Maximum is the maximum variable difference. It is never rescaled, but the exclusion option for missing values can be used.

Options

Renumber ordinal variables Use this option to specify that ordinal variables should be recoded to 1:M (see above). This is done automatically if using the Gower measure.

Weight variables Check this box to specify a weight for each clustering variable. Enter a blank-separated list of weights in this field. The weights must be nonnegative.

Missing Values There are three missing value treatments available to control how the difference between a pair of cases is calculated when the cases contain missing values.

Rescale measure in proportion causes the calculated difference to be scaled up according to the number of variable differences that cannot be calculated. For example, if there are three clustering variables and one pair cannot be calculated because the value is missing in either or both cases, the calculated value is multiplied by 3/(3-1).

Use measure without rescaling uses the value calculated from the available pairs.

Exclude case if any missing causes the distance to be missing if any variables are missing.

Output

Viewer output always includes a summary table of the silhouette data. In addition, three types of plots are available.

Histogram by cluster produces a paneled plot showing the distribution of the silhouette values for each cluster. In horizontal orientation, all the plots appear on a single row. If there are many clusters, using vertical orientation may be preferable as it produces one plot per row.

3-D bar of silhouettes by assigned and next best clusters Check this box to produce a 3-d bar chart with the assigned and next best cluster numbers on the x and z axes, respectively, and the mean silhouette score on the y axis.

3-D bar of counts Check this box to produce a 3-d bar chart with the assigned and next best clusters on the x and z axes and the percentage of cases on the y axis. The percentages sum to 100 for each assigned cluster.
