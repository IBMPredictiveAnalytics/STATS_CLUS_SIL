Статистика силуэтов для кластерного анализа

Эта процедура создает статистику силуэтов для результатов кластерного анализа, которая может быть полезна при оценке результатов кластеризации.

Для этой процедуры требуется, чтобы результаты наблюдений были в памяти, причем время выполнения пропорционально квадрату числа наблюдений. Если у вас недостает памяти и терпения для такой процедуры, рассмотрите возможность использования для нее произвольной выборки из ваших данных.

Число кластеров Выберите переменную для хранения номера участия в кластере. Обычно эти значения изменяются в диапазоне от 1 до K, где K - это количество кластеров, однако разрешены и пропуски в нумерации, которые рассматриваются как пустые кластеры.

Переменные результатов Введите имена для переменных, которые будут содержать номера следующего лучшего кластера для каждого наблюдения и для значения силуэта. Существующие значения, если они есть, будут перезаписаны.

• Диапазон значений силуэта - это интервал от -1 до 1, причем значения около 1 означают, что наблюдение хорошо размещено, а небольшие положительные или все отрицательные значения - что плохо.

Мера различия Выберите меру, которая будет использоваться для вычисления различий между наблюдениями, то есть и оценок силуэтов. Обычно такая мера согласована с мерой, используемой для вычисления кластеров, в данной процедуре не представлены все многочисленные различные меры кластеризации, доступные в пакете Statistics.

• Все переменные кластеризации должны быть числовыми, кроме меры Гауэра.

Евклидова мера - стандартное евклидово расстояние

Абсолютная разность - сумма абсолютных значений разностей переменных

Мера Минковского - корень p-ой степени из суммы p-ых степеней абсолютных разностей. Введите показатель степени для использования в дальнейшем.

Мера Гауэра различает переменные в соответствии с уровнем измерений. Количественные переменные измеряются абсолютной разностью, причем каждая разность делится на величину диапазона переменной. Порядковые переменные перенумеровываются, чтобы диапазон значений определялся от 1 до M, где M - это количество отдельных значений, а затем эти переменные рассматриваются как числовые. Разности номинальных переменных могут быть равны 0 или 1 в зависимости от равенства или неравенства значений.

Максимум - это максимальная разность переменных. Это значение никогда не перемасштабируется, но для значений отсутствия может быть сделано исключение.

Параметры

Перенумеровать порядковые переменные Используйте эту опцию для указания, что порядковые переменные должны быть перекодированы в диапазон 1:M (смотрите выше). Это делается автоматически с использованием меры Гауэра.

Переменные веса Включите этот переключатель, чтобы задать вес для каждой переменной кластеризации. Введите в этом поле список разделенных пробелами весов. Веса должны иметь неотрицательные значения.

Значения отсутствия Есть три подхода к рассмотрению значений отсутствия и к вычислению разности между парой наблюдений, когда они содержат такие значения.

Пропорциональное изменение масштаба приводит к тому, что вычисленная разность увеличивается в соответствии с количеством разностей переменных, которые нельзя вычислить. Например, если есть три переменных кластеризации и одну их пару нельзя обсчитать из-за отсутствия значения для одного или обоих наблюдений, вычисленное значение умножается на 3/(3-1).

Мера без изменения масштаба использует значение, вычисленное по доступным парам.

Исключение наблюдения при любом отсутствии значения приводит к тому, что расстояние отсутствует, если отсутствуют любые переменные.

Вывод

Вывод средства просмотра всегда содержит сводную таблицу данных силуэтов. Кроме этого, доступны три вида диаграмм.

Гистограмма по кластеру создает снабженный панелями график, показывающий распределение значений силуэтов для каждого кластера. При горизонтальной ориентации все графики выводятся в одной строке. Если кластеров много, предпочтительнее использовать вертикальную ориентацию, так как в этом случае выводится по одному графику на строку.

Трехмерная столбчатая диаграмма силуэтов по назначенным и следующим лучшим кластерам Включите этот переключатель, чтобы создать трехмерную столбчатую диаграмму, в которой по осям x и z соответственно откладывается количество назначенных и следующих лучших кластеров, а по оси y - средняя оценка силуэта.

Трехмерная столбчатая диаграмма отсчетов Включите этот переключатель, чтобы создать трехмерную столбчатую диаграмму, в которой по осям x и z соответственно откладывается количество назначенных и следующих лучших кластеров, а по оси y - процентная доля наблюдений. Для каждого назначенного кластера сумма значений процентной доли равна 100.

Дополнительные возможности

Это диалоговое окно генерирует синтаксис для команды расширения STATS CLUS SIL. Чтобы вывести справку для этой команды, введите следующую команду:

STATS CLUS SIL /HELP.

Технические требования

Для выполнения этой команды требуется Python Essentials и команда расширения STATS CLUS SIL. Можно загрузить их из SPSS Community по адресу www.ibm.com/developerworks/spssdevcentral.
